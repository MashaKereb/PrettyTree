Pretty tree
=
A script for parsing trees from a string and printing then to the terminal.
(Works like
[asciitree](http://pythonhosted.org/asciitree/))
##Input
 ```
 "(asciitree (sometimes you) (just (want to draw)) trees (in (your terminal)))"
```

##Result
```
asciitree
 +-- sometimes
 |    +-- you
 +-- just
 |    +-- want
 |    +-- to
 |    +-- draw
 +-- trees
      +-- in
           +-- your
                +-- terminal
```
